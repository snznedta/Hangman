
# Hangman

An online hangman game based on django

### Installing

After unzip the file, run:

```
bash install.sh
```

## Running the server

```
python manage.py runserver 0.0.0.0:8000

```

## Playing game
```
visit http://localhost:8000 in your browser.
```

## Built With

* [django](https://www.djangoproject.com/) - The web framework used
